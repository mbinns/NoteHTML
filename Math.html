Addition

  • If there is a floating point number then the result will be floating

Subtraction

  • If there is a floating point number then the result will be floating

Multiplication

  • If there is a floating point number then the result will be floating

Division

  • In python 3.0+ division will auto interpret as floating point
    • But in older version of python you will have to ensure you have the correct precision setup

Floor

  • Floor division is represented by '//':
    • 23/5 = 4.6 23//5 = 4
  • The result of this division will always be rounded down

Mod

  • Mod is written as '%'
  • Mod gives the remainder of the division
  • You must use a float to get results in floating point format
  • If there is no remainder then you get one

Exponent

  • Exponents are written as '**':
    • 2**5 is = 2^5
