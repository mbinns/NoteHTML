Variables

  • You can assign multiple variables the same value by doing:
    • a=b=c
  • You can assign multiple variables different values by using commas:
    • one, ,two, three = 1 , 'two', 3.0
      • Notice the 1 does not get assigned to the three variable but instead they follow the order of the commas
  • Variables don't care what data type is put in them
  • You can't assign variable names that are keywords:
    • To check keywords: 
      • import keyword
