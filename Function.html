Defining

  • the def keyword is used to define a function in python
  • Example: 
    • # function with no args or return
def function():
    print("This is a function")
function()

Returning

  • Functions can return a value in python, this does not need to be identified in the definition unlike C
  • You can return multiple values, it puts the values into a list object
  • Example: 
    • # single value return
def returning():
    return "Hello mack!"
    
result = returning()
print(result)

# multi result return
def multival():
    return "this result", 2

Arguments

  • Arguments are specified in the () of the function
  • there can be as many parameters as you want to pass:
    • arguments are pass by value by default.
  • Defaults:
    • These are the default values if they are not specified
    • Default values should be put last
  • Example: 
    • # Single argument
def arguments(a):
    print(a)

arguments("Hello mack!")

# multi args
def arguments(a,b):
    print(a)
    print(b)
    
arguments("hello", " mack!")

# default values of the function
def defaults(a, b=4,c=5):
    return a+b+c
    
result = defaults(5)

Scope

  • The scope of an argument in a function is only with in the function unless passed by reference

Nested functions

  • Functions can be nested inside of another function
  • The outer function can see inside of the function contained within it
    • The inner function can only see the stuff inside of it, not the outer functions stuff
  • You can define multiple nested function values by adding extra () to the function call to the outer
  • Example: 
    • # one def
def outer(a):
    def nested(b):
        return b * a
    a = nested(a)
    return a
print(outer(4))

# two def 4 is outer 2 is inner
def outer(a):
    def nested(b):
        return b * a
    a = nested(a)
    return a
print(outer(4)(2))

Recursion

  • A function calling its self
  • You need a base case for a recursive call
  • Examples: 
    • # recursion, solving a factorial example
def factorial(n):
    if n == 1:
        return 1
    else:
        return(n * factorial(n-1))
print(factorial(5))

Lambda

  • Lambda is a nameless function which is stored in a variable, these are used for inline functions.
  • Lambda functions can also be nested like normal functions
  • Example: 
    • f = Lambda x,y: x + y
print(f(2,3))
