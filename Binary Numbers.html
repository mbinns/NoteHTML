Operators

  • <<
    • The number on the left will be shifted to the left
    • 0001 = 2^0 = 1
    • 1 << 3
    • 1000 = 2^3 = 8
  • >>
    • The number on the left will be shifted to the right
    • 100000 = 2^5 = 32
    • 32 >> 5
    • 000001 = 2^0 = 1
  • &
    • This is if the numbers are the same it returns 1 if no returns 0
    • And compares each position of the binary representation of the number and if they match is true
    • 255 & 255
    • Example:

      • 10000000000 (1024)
      • 01111111111

  • |
    • This is an or operator, if one side is true then the expression is true
    • 10000000000 Number
    • 01111111111 Number 2
    • 11111111111 Result
    • If both sides are true then also true
    • If both sides are false then expression is false
  • ^
    • XOR operator, if one or the other is true then the expression is true. Both expressions cannot be true
