Indexing

  • You can access a specific char of a string by using an index []
  • Starts at 0 and is the length of a string
    • string[1]
    • Will provide the second character of the string
  • Reverse indexing
    • Starts indexing at 1
    • string[-1]
    • Will provide the last character in the string (this cannot start at zero because there is no -0 
  • # will provide the length of the string
string = "I am a string"
len(string)
print(string[1])
print(string[-1])
  • You cannot assign a character using string indexing

String cutting

  • This allows you to return a range of characters in a string and is similar to indexing
  • [#:#] is the format to follow
    • Examples: 
      string = "hello mack!"
string[3:7] # would return 'lo ma'
string[:3] # would return 'hell' because it starts at zero and returns to the 3 index
string[3:] #would return 'lo mack!' because it starts at the 3rd position and goes until the end

Concatenation

  • This is putting the two strings into one line
  • Uses the '+' operator to glue them together
  • Example: 
    • string = "hello "
string1 = "mack!"
string3 = string + string1 # string3 contain 'hello mack!'
  • You can repeat concatenation using the multiplier
    • Follows order of string operations
    • Example: 
      •     string = "hello "
    string1 = "mack!"
    string3 = 2 * (string + string1) # string3 contain 'hello mack!hello mack!'

Format method

  • Formatting is inserting dynamic values into a string
  • {0},{1},etc is the formatting strings
  • Variables can also be used in place of the {0} indexing method or also along side of them, BUT in the format string you must follow the indexing
  • You can also format as binary, hex, octal using {:b} or {:x} or {:o}
  • Example: 
    • # indexing example
print("today I had {0} cups of {1}".format(3,"coffee"))
# variable example
print("today I had {x} cups of {z}".format(x = 3,z = "coffee"))
# combo example notice how the index had to come first
print("today I had {x} cups of {0}".format("coffee", x = 3))
# Binary
print("today I had {:b} cups of {0}".format("coffee", x = 3))
# Hex
print("today I had {:x} cups of {0}".format("coffee", x = 3))
# octal
print("today I had {:o} cups of {0}".format("coffee", x = 3))
